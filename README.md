# Dorker
Automate Dorking With Puppeteer   

Compiles a list of Hosts and their links found from dorking image searches into either CSV or JSON format.

### Installing
`sudo npm install -g git+https://github.com/RadicalEds/Dorker.git`

### Usage
```dorker <service> <query>```

Currently only duckduckgo is supported.    

For more Information try `dorker -h`
